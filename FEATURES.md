# 功能说明

## 截图答题

- 拖动框选固定截图区域
- 使用全局快捷键记录区域两个坐标点
- 自动保存并恢复最近截图区域
- 截图时隐藏 GotIt 窗口
- 显示截图、OCR、AI 和完成状态
- 防止处理中重复启动任务
- 支持完整答题和仅 OCR 两种模式
- 保存截图与 OCR 文本
- 复制最近结果并查看处理详情

## 文字对话

- 输入文本或读取系统剪贴板
- 保留最近 50 条消息，发送时携带最近 20 条
- 将 AI 回答复制回剪贴板
- 快捷键发送剪贴板内容
- 快捷键清空上下文
- 对话上下文存放在系统用户数据目录

## 设置

- 配置 OpenAI 兼容模型、API 密钥和服务地址
- 配置温度与最大输出长度
- 设置 OCR 置信度过滤
- 设置截图延迟与自动隐藏窗口
- 控制详情弹窗和系统通知
- 修改截图全局快捷键
- 所有配置保存后立即生效

## 平台支持

- Windows：截图、全局快捷键和 Windows Toast 通知
- macOS：截图、全局快捷键和系统通知
- Linux：截图、全局快捷键和 `notify-send` 通知

部分系统需要为 Python 或终端授予屏幕录制、辅助功能或通知权限。
