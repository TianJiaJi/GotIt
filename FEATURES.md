# 功能说明

## 截图答题

### 区域管理
- 拖动框选固定截图区域
- 使用全局快捷键记录区域两个坐标点
- 自动保存并恢复最近截图区域
- 截图时隐藏 GotIt 主窗口

### 智能识别与答题
- 显示截图、OCR、AI 和完成状态
- 防止处理中重复启动任务
- 支持完整答题和仅 OCR 两种模式
- 保存截图与 OCR 文本
- 复制最近结果并查看处理详情

### 题目类型识别
- **单选题识别**：自动识别并格式化输出选项
- **多选题识别**：识别所有正确选项
- **判断题识别**：统一输出正确/错误
- **解答题识别**：支持填空、计算、简答题
- **OCR 错误纠正**：自动修正常见 OCR 识别错误
  - 字母与数字混淆（O→0，l/I→1 等）
  - 标点符号错误
  - 数学符号错误
  - 选项格式丢失

## 文字对话

- 输入文本或读取系统剪贴板
- 保留最近 50 条消息，发送时携带最近 20 条
- 将 AI 回答复制回剪贴板
- 快捷键发送剪贴板内容
- 快捷键清空上下文
- 对话上下文存放在系统用户数据目录

## 系统托盘

### 托盘功能
- 最小化到系统托盘
- 双击托盘图标显示/隐藏窗口
- 右键菜单操作：
  - 显示窗口
  - 隐藏到托盘
  - 退出程序

### 托盘通知
- 答题完成通知
- 错误提示通知
- 可在设置中控制通知开关

## 设置

### AI 配置
- 配置 OpenAI 兼容模型、API 密钥和服务地址
- 配置温度与最大输出长度
- 支持多种 LLM 提供商

### OCR 配置
- 设置 OCR 置信度过滤
- 选择 OCR 语言模式

### 截图行为
- 设置截图延迟与自动隐藏窗口
- 保存和恢复截图区域

### 界面控制
- 控制详情弹窗和系统通知
- 修改截图全局快捷键
- 所有配置保存后立即生效

## 平台支持

### Windows
- 截图、全局快捷键
- Windows Toast 通知
- 系统托盘完整支持

### macOS
- 截图、全局快捷键
- 系统通知
- 系统托盘（部分支持）

### Linux
- 截图、全局快捷键
- `notify-send` 通知
- 系统托盘支持

**权限说明**：部分系统需要为 Python 或终端授予屏幕录制、辅助功能或通知权限。
