# 截图答题工具

一个基于Python的截图答题工具，支持屏幕截图、OCR识别和AI答题功能。

## 当前功能

- ✅ 使用Tkinter创建GUI界面
- ✅ 一键截图功能
- ✅ 可自定义的全局快捷键支持（默认Alt+Shift+S）
- ✅ 快捷键设置界面
- ✅ 配置文件自动保存
- ✅ 自动保存截图到项目根目录

## 使用方法

1. 运行程序：`python main.py`
2. 两种截图方式：
   - **方式1**：点击窗口中的"截取屏幕"按钮
   - **方式2**：按下快捷键（默认：`Alt+Shift+S`，全局有效，窗口最小化时也可用）
3. **自定义快捷键**：
   - 点击"快捷键设置"按钮
   - 在对话框中选择修饰键（Ctrl、Alt、Shift及其组合）
   - 选择按键（A-Z、0-5等）
   - 点击"确定"保存设置
4. 截图将自动保存到项目根目录，文件名格式：`screenshot_YYYYMMDD_HHMMSS.png`

## 配置文件

程序会自动在项目根目录创建 `config.ini` 文件保存用户的快捷键配置：
```ini
[hotkey]
modifiers = alt+shift
key = s
```

## 依赖安装

```bash
pip install -r requirements.txt
```
