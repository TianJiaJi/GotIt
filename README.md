# 截图答题工具

一个智能化的截图答题工具，支持区域截图、OCR文字识别等功能。

## 功能特性

- 📸 **智能截图**
  - 支持全屏截图
  - 支持自定义区域截图
  - 快捷键截图支持

- 🔲 **区域设置**
  - 界面按钮设置：点击按钮 → 移动鼠标 → 点击确认
  - 快捷键设置：ALT+CTRL+1/2 直接记录当前鼠标位置
  - 区域可视化显示

- 🔤 **OCR识别**
  - RapidOCR + onnxruntime 高精度识别
  - 自动保存识别结果为文本文件
  - 中英文混合识别支持

- ⌨️ **快捷键支持**
  - 可自定义截图快捷键
  - 实时快捷键状态显示
  - 区域设置快捷键：ALT+CTRL+1/2

## 使用方法

### 1. 启动程序
```bash
python main.py
```

### 2. 设置截图区域（可选）
**方式一：界面按钮**
1. 点击"① 设置点1"按钮
2. 移动鼠标到目标位置（如题目左上角）
3. 点击鼠标左键确认
4. 同样方式设置点2

**方式二：快捷键**
1. 按 `ALT + CTRL + 1` 设置点1
2. 按 `ALT + CTRL + 2` 设置点2

### 3. 截图并识别
- 点击"截取屏幕"按钮或按自定义快捷键
- 程序自动截图并进行OCR识别
- 识别结果会自动弹出显示

### 4. 管理功能
- **清除区域**：恢复全屏截图
- **快捷键设置**：自定义截图快捷键
- **ESC键**：取消区域设置模式

## 输出文件

每次截图会生成两个文件：
- `screenshot_YYYYMMDD_HHMMSS.png` - 截图文件
- `screenshot_YYYYMMDD_HHMMSS.txt` - OCR识别结果

## 模块说明

### config.py - 配置管理
- 管理程序配置文件
- 快捷键配置的读取和保存
- 默认配置创建

### ocr.py - OCR识别
- RapidOCR引擎初始化
- 图片文字识别
- 识别结果保存

### screenshot.py - 截图功能
- 全屏截图
- 区域截图
- 文件保存管理

### region.py - 区域设置
- 截图区域管理
- 鼠标位置记录
- 区域信息获取

### hotkey.py - 快捷键管理
- 全局快捷键监听
- 按键状态管理
- 快捷键回调处理

### ui.py - 用户界面
- 主窗口界面
- 快捷键设置对话框
- OCR结果显示对话框

### main.py - 主程序
- 程序入口
- 各模块集成
- 事件处理协调

## 技术栈

- **界面框架**: tkinter
- **OCR引擎**: RapidOCR + onnxruntime
- **图像处理**: PIL (Pillow)
- **快捷键**: pynput
- **配置管理**: configparser
